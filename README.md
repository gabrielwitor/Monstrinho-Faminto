# Monstrinho Faminto

</p>

Monstrinho Faminto é um jogo educativo focado na alfabetização infantil, inteiramente desenvolvido na engine Godot 4. O objetivo principal do jogo é incentivar a leitura e a formação de palavras de maneira lúdica e interativa. Durante a jogabilidade, os jogadores precisam arrastar os blocos contendo as sílabas corretas para formar palavras e, assim, alimentar o monstrinho.

## Características

- Desenvolvido em Godot 4.
- Foco educacional voltado para crianças em fase de alfabetização.
- Mecânica simples e intuitiva de "arrastar e soltar" (Drag and Drop).
- Aprendizado interativo através de tentativa e recompensa visual.

## Controles

- **Mouse:** Utilize o botão esquerdo do mouse para clicar, segurar e arrastar os blocos de sílabas até o local correspondente para formar a palavra.

## Como Baixar e Jogar

Para quem deseja apenas jogar, não é necessário baixar o código-fonte ou instalar a engine Godot. Basta fazer o download da versão pré-compilada.

1. Acesse a página de [Releases](https://github.com/gabrielwitor/Monstrinho-Faminto/releases/tag/v1.0) do repositório.
2. Na versão mais recente disponível, desça a página até encontrar a seção "Assets" no final da nota de atualização.
3. Baixe o arquivo compactado que corresponde ao seu sistema operacional (Windows, Linux ou macOS).
4. Extraia o conteúdo do arquivo baixado em uma pasta de sua escolha no computador.
5. Execute o arquivo principal do jogo para iniciar a partida.

## Como Executar a Partir do Código-Fonte

Caso tenha interesse em estudar o código, modificar o jogo ou contribuir com o desenvolvimento, você pode abrir o projeto diretamente no editor da Godot:

1. Faça o download e instale o [Godot Engine 4](https://godotengine.org/download).
2. Clone este repositório para o seu computador ou faça o download como arquivo ZIP.
3. Abra o gerenciador de projetos da Godot Engine, clique em "Importar" e selecione o arquivo `project.godot` localizado na pasta raiz deste projeto.
4. Clique em "Executar" no editor (ou pressione a tecla F5) para testar o jogo.
